# 邮箱授权码配置说明

## 什么是授权码?

授权码是用于第三方客户端登录邮箱的专用密码,用来代替你的邮箱登录密码,提供更安全的访问方式。

---

## 常见邮箱授权码获取方法


### 📮 QQ邮箱

1. 登录 [QQ邮箱网页版](https://mail.qq.com)

![](https://files.mdnice.com/user/151096/e89bf4fd-b3cc-4c75-bd86-5e5adabfbb04.png)

2. 点击顶部 **设置** → **账户**

![](https://files.mdnice.com/user/151096/7226ce00-0c16-4981-8b9a-f861ce003546.png)

3. 点击，选择**安全设置**，找到 **POP3/IMAP/SMTP/Exchange/CardDAV/CalDAV服务**

![](https://files.mdnice.com/user/151096/25da52f2-4cc6-4c9b-9f98-9b7138e72dc3.png)


4. 开启 **POP3/SMTP服务** 或 **IMAP/SMTP服务**
![](https://files.mdnice.com/user/151096/dfd68503-1abd-4bba-b585-23ffa0b1dd4c.png)

5. 点击 **生成授权码**

![](https://files.mdnice.com/user/151096/7e1f9b76-5cbf-498b-9e18-e6ddf844f335.png)

6. 选择其它方式接收短信验证。
![](https://files.mdnice.com/user/151096/d1d8c369-3239-4faa-ab88-813c5f14bd45.png)

7. 按照短信验证后,系统会显示一个16位的授权码

![](https://files.mdnice.com/user/151096/09f0734e-1aad-4520-9fc6-98391ae182ba.png)

8. **重要**: 复制并妥善保存这个授权码,它只会显示一次!

### 📧 163邮箱

1. 登录 [163邮箱网页版](https://mail.163.com)
2. 点击顶部 **设置** → **POP3/SMTP/IMAP**
3. 开启 **POP3/SMTP服务** 或 **IMAP/SMTP服务**
4. 点击 **客户端授权密码**
5. 输入手机验证码或密保问题答案
6. 系统会生成一个授权码
7. 复制并保存授权码

---

### 📬 126邮箱

1. 登录 [126邮箱网页版](https://mail.126.com)
2. 点击顶部 **设置** → **POP3/SMTP/IMAP**
3. 开启相应服务(POP3/SMTP 或 IMAP/SMTP)
4. 点击 **客户端授权密码**
5. 验证身份后获取授权码
6. 复制并保存

---

### 📨 Gmail (谷歌邮箱)

**前提**: 需要开启两步验证

1. 登录 [Google账户](https://myaccount.google.com)
2. 进入 **安全性**
3. 找到 **两步验证** 并开启
4. 开启后,在同一页面找到 **应用专用密码**
5. 选择应用类型(如"邮件")和设备
6. 点击 **生成**
7. 系统会显示16位的应用专用密码
8. 复制并保存

---

### 📮 Outlook/Hotmail

**注意**: Microsoft已逐步停用应用密码,建议使用OAuth2.0认证

如需使用应用密码:
1. 登录 [Microsoft账户安全页面](https://account.microsoft.com/security)
2. 开启 **两步验证**
3. 进入 **高级安全选项**
4. 找到 **应用密码** → **创建新的应用密码**
5. 输入名称(如"邮件客户端")
6. 复制显示的密码

## ⚠️ 重要提示

1. **授权码 ≠ 邮箱密码**: 授权码是专门用于第三方客户端的,不是你的邮箱登录密码
2. **只显示一次**: 大多数邮箱的授权码只会显示一次,请立即复制保存
3. **安全保管**: 授权码等同于邮箱密码,请妥善保管,不要泄露给他人
4. **可以重置**: 如果忘记或泄露,可以删除旧授权码并重新生成
5. **多个授权码**: 可以为不同的应用生成多个授权码,便于管理

## 📋 使用授权码配置邮箱

配置时的基本信息:

- **邮箱地址**: 你的完整邮箱地址 (如: example@qq.com)
- **密码/授权码**: 填写刚才获取的授权码(不是邮箱登录密码!)
- **SMTP服务器**: 
  - QQ邮箱: `smtp.qq.com` (端口: 465 或 587)
  - 163邮箱: `smtp.163.com` (端口: 465 或 994)
  - 126邮箱: `smtp.126.com` (端口: 465)
  - Gmail: `smtp.gmail.com` (端口: 465 或 587)
  - Outlook: `smtp-mail.outlook.com` (端口: 587)

## ❓ 常见问题

**Q: 为什么找不到授权码选项?**
A: 需要先开启POP3/SMTP或IMAP服务,开启后才会出现授权码生成选项。

**Q: 授权码忘记了怎么办?**
A: 可以删除旧的授权码,重新生成一个新的。

**Q: 可以用邮箱密码代替授权码吗?**
A: 大部分邮箱服务商为了安全,已经禁止使用邮箱密码直接登录第三方客户端,必须使用授权码。

**Q: 授权码有有效期吗?**
A: 一般没有有效期限制,除非你手动删除或重置。
